Decent Holograms

Decent Holograms is an open source that has a bunch of features. Some features include adding lines animations using color. It even has a simple config file. To create a hologram type the command /holo create {name} {content}, it's literly so easy to get started you can dowload this plugin on Spigot right now.

LuckPerms
Luckperms is an easy to use database that allows you to create custom teams like: owner, helper, and mods. The best part about Luckperms is you can add any commands that only that certain rank can use. If they use a command that they are not allowed to use it will not show them the command. It will also say "You do not have permission to use this command." this plugin is availible at LuckPerms.net
EssentialsX
EssentialsX is one of the most incredible plugins we have used on our server. The reason of this is because it make the regular minecraft commands so much simpler and easer to use. It also adds custom commands to make the game more fun too. Like /kittycannon or /nuke. It's a great plugin that you should use. EssentialsX is availible at essentialsx.net
Citizens 2
Citizens 2 is an npc plugin that allows you to create npcs for sending people to different game modes, or making quests and a lot more. The Citizens 2 plugin is a paid plugin on the Spigot website. It is so simple to use and other servers like Hypixal or ChaseCraft use this plugin. The reason we chose to use this plugin is bacause we have a bunch of things ready to be released for all of you so keep an eye on that.
